# Task Manager CLI

Sistema de gerenciamento de tarefas em linha de comando desenvolvido em Python.
O projeto permite adicionar, remover, visualizar e salvar tarefas utilizando um arquivo JSON para persistência de dados.

## Funcionalidades

* Adicionar novas tarefas
* Remover tarefas pelo número
* Visualizar lista de tarefas
* Limpar todas as tarefas
* Salvar tarefas em arquivo JSON
* Carregar tarefas salvas

## Tecnologias Utilizadas

* Python 3
* JSON (persistência de dados)

## Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/task-manager-cli.git
```

2. Acesse a pasta do projeto:

```bash
cd task-manager-cli
```

3. Execute o programa:

```bash
python main.py
```

## Como Usar

Ao executar o programa, será exibido um menu com opções para:

* Adicionar tarefas
* Remover tarefas
* Limpar lista de tarefas
* Salvar tarefas
* Carregar tarefas
* Sair

Basta digitar a opção desejada e seguir as instruções no terminal.
